## Conventions / Gotchas

1. **Bun-only** — The `packageManager` field pins `bun@1.3.14`. Do not use `npm` or `yarn`. All scripts run via `bun run <script>`.
2. **No semicolons, single quotes** — Enforced by Prettier (`semi: false`, `singleQuote: true`).
3. **Zero-tolerance linting** — `eslint . --max-warnings=0` means any warning or error fails CI.
4. **Codegen required before build** — `generated/` is not checked in. Run `bun run codegen` first, then `bun run build`.
5. **Matchstick test version pinned** — Tests use `--version 0.6.0` (the `matchstick-as` version in `devDependencies`). Running `graph test` without the version flag may pull a different version.
6. **Coverage gating** — `bun run test:coverage` runs tests with `--coverage` and parses the output for `Global test coverage:` / `Test coverage:`; it exits non-zero below the threshold defined in `scripts/check-coverage.mjs`.
7. **Subgraph name is `nifty-league-sepolia`** — This is the dev/sepolia variant. The deploy target is Graph Studio at `https://api.studio.thegraph.com/deploy/`.
8. **Local dev node** — `create-local`/`remove-local`/`deploy-local` point at `localhost:8020` (Graph node) and `localhost:5001` (IPFS). Requires a local Graph node running.
9. **Immutable event entities vs. mutable domain entities** — `Approval`, `Transfer`, `Paused`, etc. are `immutable: true`; `Character`, `Owner`, `Contract` are `immutable: false`. `TraitMap` is `immutable: true` (created once per token).
10. **No README** — There is no `README.md` in this repo. The `subgraph.yaml` `description` field says "NiftyDegen Sepolia NFTs from NiftyLeague".
